Data Preprocessing

This app analizes and cleans dataset as a first step before performing machine learning. Data preprocessing is very importan since most of the datasets in the world are dirty, having null values, categorical data or outliers. Cleaning all this makes data much more reliable and thus, better models will be created employing machine learning and deep learning methodologies. Also, the dataset is splitted into train and test sets.
