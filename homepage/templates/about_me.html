Projects

  • A machine learning-based scoring function for drug design: The aim of this project was to create a machine learning model for finding novel drugs for skin-related decesases in collaboration with the pharma. Methods: Neural networks, support vector machines, random forests, gradient boosting and topological data analysis. Tools: Scikit-learn, Tensoflow, Pandas, Keras, XGboost.

  • Enzyme promiscuity and activity using artificial intelligence: The aim of this project was to build two different type of models. The first model was to predict the how promiscous is an esterase (a type of enzyme) using a dataset given by an experimental group. The second model was about the activity of esterases. These models were constructed performing random and grid search over different feature engineering, hyperparameters and machine learning methods pipelines. The pipeline with the best cross-validation values were used for testing. Methods: Neural networks, support vector machines, random forests, gradient boosting, algebraic topology and Monte-Carlo sampling. Tools: Scikit-learn, Tensoflow, Pandas, Keras, XGboost, Gromacs, Glide.
